api.ordering.version0 module¶

API interface for placing and viewing orders.

Any methods exposed through this interface are intended to be consumed by end users (publicly). The module should be the pure interface for the api functions. Don’t import or include any implementation specific items here, just logic. Implementations are touched through the registry.
